频的舒适性与稳定性，并充分使用了易灵思 RISC-V 核与电脑上位机通信进行用户的反馈。

### 上手指南

为了便于用户使用我们的产品，我们做了精美易使用的人机交互界面，用户选择正确的串口后即可打开串口发送功能并通过下拉菜单栏选择合适的传输波特
率与放大或缩小模式。

###### 开发前的配置要求

1. 易灵思 Ti60F225
2. Sapphire SOC
